<h2>📌 Sobre o Projeto</h2>

<p>
O <strong>GlowUp</strong> é uma aplicação web desenvolvida com arquitetura cliente-servidor, permitindo o gerenciamento de usuários, serviços e agendamentos.
</p>

<ul>
<li>Cadastro de clientes e profissionais</li>
<li>Agendamento de serviços</li>
<li>Busca com filtros</li>
<li>Integração com banco de dados relacional</li>
<li>Autenticação com JWT</li>
</ul>

<hr>

<h2>🚀 Funcionalidades</h2>

<h3>👤 Clientes</h3>
<ul>
<li>Cadastro e login</li>
<li>Busca de serviços com filtros</li>
<li>Agendamento online</li>
</ul>

<h3>🏢 Profissionais</h3>
<ul>
<li>Cadastro e gerenciamento de perfil</li>
<li>Gerenciamento de serviços e valores</li>
<li>Controle de agendamentos</li>
</ul>

<hr>

<h2>🛠️ Tecnologias Utilizadas</h2>

<ul>
<li><strong>Frontend:</strong> HTML5, CSS3, JavaScript</li>
<li><strong>Backend:</strong> PHP</li>
<li><strong>Banco de Dados:</strong> MySQL</li>
<li><strong>Autenticação:</strong> JWT (JSON Web Token)</li>
<li><strong>Ferramentas:</strong> Git, GitHub, Postman</li>
</ul>

<hr>

<h2>🔧 Arquitetura</h2>

<ul>
<li>Arquitetura cliente-servidor</li>
<li>Backend em PHP responsável pela lógica de negócio</li>
<li>Comunicação via requisições HTTP</li>
<li>Estrutura baseada em API REST</li>
<li>Integração com banco de dados MySQL</li>
</ul>

<hr>

<h2>🔐 Autenticação</h2>

<ul>
<li>Autenticação baseada em tokens JWT</li>
<li>Geração de token após login do usuário</li>
<li>Token enviado via header <strong>Authorization (Bearer Token)</strong></li>
<li>Validação do token no backend para acesso a rotas protegidas</li>
</ul>

<hr>

<h2>🌐 API</h2>

<ul>
<li>Endpoints para login e cadastro de usuários</li>
<li>Endpoints para gerenciamento de serviços e agendamentos</li>
<li>Validação de requisições no backend</li>
</ul>

<hr>

<h2>🗄️ Banco de Dados</h2>

<ul>
<li>Modelagem relacional com MySQL</li>
<li>Operações CRUD (Create, Read, Update, Delete)</li>
<li>Estrutura voltada para usuários, serviços e agendamentos</li>
</ul>

<h2>🚀 Como executar o projeto</h2>

<h3>📋 Pré-requisitos</h3>
<ul>
<li>PHP 8.0 ou superior</li>
<li>MySQL</li>
<li>Servidor local (XAMPP, WAMP ou Apache)</li>
</ul>

<h3>⚙️ Passo a passo</h3>

<pre>
git clone https://github.com/matpssnt/GlowUp.git
</pre>

<ol>
<li>Importe o banco de dados MySQL (arquivo .sql)</li>
<li>Configure as credenciais no arquivo de conexão (ex: config.php)</li>
<li>Inicie o servidor local</li>
<li>Acesse no navegador:</li>
</ol>

<pre>
http://localhost/GlowUp
</pre>
